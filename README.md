# 🏛️ Magister Linguae: Latin Translator & Grammar Analyzer

**Magister Linguae** adalah aplikasi asisten digital berbasis AI yang dirancang khusus untuk guru dan siswa bahasa Latin. Aplikasi ini tidak hanya menerjemahkan teks, tetapi juga memberikan analisis tata bahasa (parsing) mendalam, penjelasan sintaksis, dan konteks sejarah secara otomatis.

---

## ✨ Fitur Utama
- **Terjemahan Akurat:** Menerjemahkan teks Latin ke Bahasa Indonesia dengan nuansa sastra yang tepat.
- **Analisis Gramatikal (Parsing):** Membedah setiap kata (Kata Benda: Kasus, Gender, Deklinasi; Kata Kerja: Tense, Mood, Person, Number).
- **Penjelasan Sintaksis:** Mendeteksi struktur kompleks seperti *Ablative Absolute*, *Gerund*, atau *Indirect Statements*.
- **Konteks Sejarah:** Memberikan informasi latar belakang jika teks merupakan kutipan dari tokoh klasik (Cicero, Caesar, Virgil, dll).
- **Antarmuka Modern:** Desain responsif menggunakan Tailwind CSS dengan estetika "Classical Modern".

---

## 🚀 Teknologi yang Digunakan
- **Frontend:** HTML5, Tailwind CSS, JavaScript (Fetch API).
- **Backend:** Node.js, Express.js.
- **AI Engine:** OpenAI API (GPT-4o / GPT-4o-mini).
- **Library Lain:** `dotenv` (Security), `cors` (Cross-Origin Resource Sharing).
