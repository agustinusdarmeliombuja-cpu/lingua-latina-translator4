Hapus Latar Belakang?

Biasanya gambar dengan latar belakang transparan memberikan hasil terbaik. Kami dapat menghapus latar belakang gambar secara otomatis. Apakah Anda ingin melihat pratinjaunya?

Anda dapat mengatur warna latar belakang ikon aplikasi setelahnya.

Asli

Tanpa Latar Belakang

Harap unggah gambar persegi untuk menghindari distorsi. Jika memungkinkan, gunakan gambar dengan latar belakang transparan.
Ukuran yang disarankan: 1024x1024px, Ukuran maksimum: 5000x5000px

Batas unggahan file adalah 100MB. File berukuran besar dapat memperlambat pemuatan halaman.

Nama file yang diunggah tidak sesuai harapan. Ini bisa berarti salah satu dari hal-hal berikut:

  • Anda memilih file yang salah.
  • Anda memasukkan informasi yang salah pada kolom terkait. Misalnya, jika Anda mengunggah file kunci tetapi memasukkan ID kunci yang salah pada kolom sebelumnya.
  • Anda (atau perangkat Anda) telah mengganti nama file sebelum mengunggahnya.

Mohon periksa kembali input Anda. Jika Anda yakin telah melakukan semuanya dengan benar, Anda dapat mengabaikan pesan ini. Jika Anda ragu atau menduga ini mungkin kesalahan dalam panduan ini, silakan hubungi kami!

Mengambil favicon
Kami tidak dapat menemukan favicon untuk situs web Anda. Silakan unggah gambar sebagai gantinya.

Warna ini akan ditampilkan di belakang ikon aplikasi Anda.
